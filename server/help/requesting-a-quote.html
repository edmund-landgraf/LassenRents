Requesting a quote

The quote flow captures container size, contact details, delivery address, timeline, and notes for the office.

Walkthrough

  1. Open the public quote form from the main site or a container card.
  2. Select a container size and interest areas such as Containers, Delivery, Modifications, or Trucking.
  3. Submit the form to create a browser-stored demo lead.
  4. Open the admin lead queue and review the new request.
  5. Convert the lead into the rental wizard when the availability match looks good.

Demo note: quote submissions are stored in browser local storage so they appear immediately in the admin lead queue on the same machine.
